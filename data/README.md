# Local data

Medical image data is intentionally excluded from Git.

Place the restructured VerSe 2020 release here. A typical matching pair is:

```text
data/verse20/
├── rawdata/sub-verseXXX/
│   └── sub-verseXXX_dir-orient_ct.nii.gz
└── derivatives/sub-verseXXX/
    └── sub-verseXXX_dir-orient_seg-vert_msk.nii.gz
```

Use public VerSe data only. Do not copy proprietary or patient-derived company data into this repository.

Official source: https://github.com/anjany/verse
